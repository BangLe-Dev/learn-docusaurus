# API CRUD Documentation

## Base URL

```
https://api.example.com/v1
```

## Authentication

All requests must include an API key in the headers:

```http
Authorization: Bearer YOUR_API_KEY
```

## Error Handling

| Status Code | Meaning |
|------------|---------|
| 400 | Bad Request |
| 401 | Unauthorized |
| 403 | Forbidden |
| 404 | Not Found |
| 500 | Internal Server Error |
